## Consumo de API

Este projeto foi desenvolvido em sala de aula com o objetivo de aprender como realizar o consumo de APIs.

A aplicação realiza requisições a uma API pública para buscar imagens e informações disponíveis em um banco de dados externo e exibi-las em uma página web desenvolvida como frontend.

Com isso, foi possível praticar conceitos como requisições HTTP, manipulação de dados recebidos da API e exibição dinâmica dessas informações na interface do usuário.
